# 🖼️ DECODELABS_GENAI_TASK3

## 🚀 Multimodal Image Prompt Generator

This project generates detailed and creative AI image prompts from user descriptions using the **Groq API** and **Python**. It helps users create high-quality prompts for AI image generation models.

---

## ✨ Features

- 🎨 Generate detailed AI image prompts
- ⚡ Powered by Groq API
- 🔐 Secure API key management using `.env`
- 💻 Simple command-line interface
- 📝 Easy to customize and extend

---

## 🛠️ Technologies Used

- 🐍 Python
- ⚡ Groq API
- 📦 python-dotenv

## ▶️ How to Run

### 1️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 2️⃣ Add your Groq API Key

Create a `.env` file and add:

```env
GROQ_API_KEY=your_api_key_here
```

### 3️⃣ Run the project

```bash
python app.py
```
